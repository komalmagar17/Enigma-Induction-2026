# QuickNotes Web - Enigma Induction 2026

A modern, responsive notes-taking application inspired by Google Keep, built using React.

## Features
- **Create, Edit, and Delete Notes**: Manage your thoughts easily.
- **Google Keep Style**: Card-based staggered layout with dynamic colors.
- **Search Functionality**: Quickly find your notes by title or content.
- **Pinning**: Keep important notes at the top.
- **Dark Mode**: Support for system-wide dark theme.
- **Local Persistence**: Notes are saved in the browser's local storage.
- **Splash Screen**: Beautiful entry animation.

## Tech Stack
- **Frontend**: React (v18)
- **Styling**: Custom CSS & Material UI concepts
- **Storage**: Browser LocalStorage

## How to Run
Simply open the `index.html` file in any modern web browser. No local server or build process is required as it uses CDN-based React and Babel.

## Design Philosophy
- **Minimalism**: Clean typography and generous whitespace.
- **Vibrancy**: Inspired by Google Keep's soft color palette.
- **Responsiveness**: Works on both desktop and mobile browsers.
