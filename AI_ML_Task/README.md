# Text Summarizer using Hugging Face

This project implements a text summarization program using a pre-trained model from Hugging Face's `transformers` library.

## Objective
Build a program that takes a long paragraph as input and generates a concise, meaningful summary.

## Tools Used
- Python
- Hugging Face Transformers
- PyTorch

## How to Run
1. Install the required libraries:
   ```bash
   pip install transformers torch
   ```
2. Run the summarization script:
   ```bash
   python summarizer.py
   ```

## Example Input
"Artificial Intelligence is transforming many industries by allowing machines to perform tasks that normally require human intelligence. It is widely used in healthcare, finance, robotics, and automation."

## Example Output
"Artificial Intelligence is widely used in healthcare, finance, robotics, and automation. It allows machines to perform tasks that normally require human intelligence and is transforming many industries."
